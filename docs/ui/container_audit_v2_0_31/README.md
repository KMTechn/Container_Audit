# Container_Audit v2.0.31 승인 UI 방향

`Inspection_worker v2.0.49`의 작업 집중형 구조를 기준으로 Option 1을 승인 방향으로 고정한다.

- 중앙: 단계, 다음 행동, 목표 진행도, 스캔 입력, 단일 경고, 현재 트레이 스캔 목록
- 중앙 하단: 상태와 무관하게 같은 위치에서 현재 트레이의 전체 스캔 목록을 표시
- 중앙 작업 버튼: `마지막 스캔 취소 → 트레이 보류 → 트레이 제출 → 운영 작업`
- 좌측: 현재 품목·작업자·보류 트레이
- 우측: 현재 상태·소요 시간·마지막 정상 스캔·다음 행동과 보조 통계
- 오류와 `OPERATOR_REVIEW`: 목록과 마지막 정상 스캔을 보존하며 레이아웃을 바꾸지 않음
- 리셋·현품표 교체·제품 교환: `운영 작업` 메뉴에서 제공

승인 목업은 `approved-option-1.png` 한 장만 보존한다. 상태별 반복 캡처와 자동 검증 증거는 Git 추적 대상 밖의 `tmp/`에서 관리한다.
